# 镜头式写作法

忘掉全知视角，你是跟在主角身后的摄像机。读者只能看到主角看到的东西。

## 核心原则

1. **只写能看到/听到/闻到/摸到的**：不写「他心里想」「他感到」
2. **对话即剧情**：每句对话都要推进剧情、塑造人物、或传递信息
3. **动作代替描述**：「他很紧张」→「他的手在抖」
4. **手机屏幕优先**：短句、频繁换行、每段不超过 3 句

## 禁用清单

### 禁用视角
- ❌ 全知叙述：「此时的他还不知道，三天后…」
- ❌ 上帝视角：「如果她知道真相…」
- ❌ 心理分析：「他内心深处其实…」

### 禁用词汇
仿佛、好像、犹如、一丝、一抹、深吸一口气、缓缓、不禁、眼中闪过、嘴角勾起、眉头微皱、不容置疑、不易察觉

### 禁用句式
- 「...，带着...」（如：他说，带着一丝无奈）
- 「XX说道」→ 用动作神态替代
- 「他/她知道...」→ 直接展示，不告诉
- 「像XX一样的XX」→ 陈词滥调的比喻

## 替代技巧

| 禁用 | 替代 |
|------|------|
| 他感到紧张 | 他的手在抖 |
| 她很伤心 | 她转过身，肩膀微微颤动 |
| "好的。"他说道 | "好的。"他点了根烟 |
| 他深吸一口气 | 他把烟摁灭在烟灰缸里 |
| 她眼中闪过一丝惊讶 | 她的手停在了半空 |
| 他不禁笑了 | 他的嘴角翘了一下 |

## 对话写作规范

### 每句对话必须满足至少一个：
1. 推进剧情（透露新信息、做出决定、发生冲突）
2. 塑造人物（展现性格、背景、价值观）
3. 制造张力（威胁、隐瞒、试探）

### 对话格式
- 用动作打断长对话
- 每次说话不超过 3 句
- 用神态/动作代替「他说」「她说」

### 好的对话
```
"钱呢？"

王少往椅背一靠，翘起二郎腿。"什么钱？"

"上个月的三十万。"林风往前一步。

王少的手指敲了敲桌面。"我说了吗？我不记得了。"
```

### 差的对话
```
"你还欠我三十万块钱，是上个月借的，答应这个月还的。"林风说道。

"哦，那个钱啊，我最近手头有点紧，能不能再等等？"王少不好意思地说道。

"不行，我已经等了一个月了。"林风坚定地说道。
```
