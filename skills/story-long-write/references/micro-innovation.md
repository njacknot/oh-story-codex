# 微创新手法

不是原创，是在已有套路的基础上做差异化。读者要的是熟悉的惊喜。

## 灵感源头

| 阶段 | 来源 | 方法 |
|------|------|------|
| 前期 | 标签系统 | 热门标签的交叉组合 |
| 中期 | 社交媒体 | 热搜、评论区、弹幕的高频话题 |
| 后期 | 电影/游戏 | 视觉化叙事、交互式体验 |

## 标签系统

每个标签评估三个维度：
- **兼容度**：这个标签能和多少其他标签组合
- **影响度**：这个标签对故事的影响有多深
- **新鲜度**：这个标签多久没被大量使用了

选标签的策略：高兼容 + 中等影响 + 高新鲜度 = 最佳切入点

## 五种微创新手法

### 1. 精炼法
把已有套路做到极致。别人写打脸只写一巴掌，你写三巴掌还层层递进。

### 2. 升级法
保持框架不变，把具体元素升级。都市文里的「公司」升级成「跨国集团」，「富二代」升级成「财阀继承人」。

### 3. 加料法
在已有框架里加入新元素。重生文 + 规则怪谈，追妻文 + 职场斗争。

### 4. 反套路法
反过来写。别人写男主追女主，你写女主追男主但男主是装的。

### 5. 组合法
把两个不相干的套路组合。种田文 + 修仙，系统文 + 悬疑。

## 冲突网络设计

不要只靠单个矛盾推动故事，要构建多层冲突网络：

### 纵向矛盾（上下级）
- 控制与服从
- 压制与反抗
- 师徒/君臣/上下级关系

### 横向矛盾（平级）
- 理念冲突
- 资源争夺
- 同行/情敌/竞争对手

### 交叉矛盾
- 多个矛盾交叉产生新矛盾
- A 和 B 有矛盾，B 和 C 有矛盾，A 和 C 因此也产生矛盾

好的长篇小说至少要有 3 层矛盾同时运作。
