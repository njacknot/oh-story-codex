# 拆文实例（3 个完整案例）

用这些案例学习拆文方法。注意每个案例的结构、情绪标记和钩子设计。

## 案例 1：现代虐恋（盲女保镖 × 霸总）

### 拆解模板

⭐ **导语**
- 背景：女主是盲人保镖，救过男主，男主承诺照顾她一辈子
- 虐点：女主发现男主和女配出轨，但为了报恩一直忍耐
- 反转：女主其实是假盲，她看得见一切背叛
- 钩子：① 女主能看见但没人知道 ② 她只给自己 7 天时间离开

1️⃣ **起**
- 女主在厨房听到男主打电话约女配
- 她假装不知道，端汤出去
- 男主接过汤时手机弹出女配消息，女主「碰巧」看到屏幕
- 虐点：男主在她面前毫无顾忌

2️⃣ **承 + 转1**
- 女主发现怀孕，想告诉男主
- 到男主公司，撞见男主给女配庆生
- 转折：她看到男主送女配的项链，是当年她送给男主的定情物
- 虐点升级：信物被转赠

3️⃣ **承**
- 女主沉默回家，收拾行李
- 男主回来，问她眼睛有没有好一点
- 她说「好多了，什么都能看见了」——双关，男主没听懂

4️⃣ **承 + 转2**
- 男主生日宴会，女配以女伴身份出席
- 女主作为「保镖」在角落站着
- 转折：女主当众宣布离婚，递上签好的协议
- 爽点：全场震惊

5️⃣ **合**
- 女主离开后，男主发现她留下的日记
- 日记记录了她「看见」的每一次背叛
- 男主崩溃，但女主已经消失
- 爽点 + 悬念：读者知道女主看得见，男主不知道

## 案例 2：古代虐恋（庶女 × 将军）

### 拆解模板

⭐ **导语**
- 背景：庶女与将军秘密相恋三年，将军承诺战后娶她
- 虐点：将军凯旋却带回正妻——是嫡姐
- 反转：庶女发现自己只是嫡姐的「练手工具」
- 钩子：① 庶女能否从贱籍翻身到龙床 ② 将军会后悔到什么程度

1️⃣ **起**
- 庶女在后院刺绣，听到将军回城的消息
- 她满心期待地准备嫁衣
- 门口传来嫡姐的声音：「夫君，你回来了」

2️⃣ **承 + 转1**
- 庶女被迫给嫡姐敬茶
- 转折：偷听到将军和嫡姐对话——「不过是拿她练练手，你别介意」
- 虐点：三年的感情只是工具

3️⃣ **承**
- 庶女表面顺从，暗中接近太子的宴席
- 用才艺引起太子注意
- 虐中蓄力：隐忍中谋划

4️⃣ **承 + 转2**
- 将军发现庶女被太子纳入府中，震惊
- 转折：庶女当众揭穿将军当年是靠嫡姐上位
- 爽点：身份反转，将军颜面尽失

5️⃣ **合**
- 庶女成为太子侧妃，后成正妃
- 将军在战场上被人追杀，无人救援
- 结局：将军临死前看到庶女的画像，终于后悔
- 爽点：迟来的后悔比死更难受

## 案例 3：权谋公主（长公主 × 四位青梅竹马）

### 拆解模板

⭐ **导语**
- 背景：长公主有四位青梅竹马的未婚夫，但她心里只有那个不爱她的将军
- 虐点：她选了最没权势的那个，放弃了一直默默守护的将军
- 反转：她选的那个才是害她母后的人
- 钩子：① 公主放弃真爱会怎样 ② 被放弃的人如何反击

1️⃣ **起**
- 抽签决定婚姻，公主表面顺从
- 她暗中做了手脚，让最弱的那个中签
- 读者知道她在保护将军

2️⃣ **承 + 转1**
- 被选中的驸马开始展露野心
- 转折：发现驸马与敌国通信
- 虐点：公主以为自己在保护人，实际养虎为患

3️⃣ **承**
- 公主被软禁，将军在外征战不知情
- 公主开始反击，利用之前的暗线传递消息
- 虐中蓄力

4️⃣ **承 + 转2**
- 将军回朝，发现公主被囚
- 转折：公主早已安排好反击计划，她不是等救援，是在等时机
- 爽点：公主从来不是弱者

5️⃣ **合**
- 驸马叛变被镇压，公主亲临城楼
- 将军问她为何当年不选自己
- 公主：「因为只有你不在我身边，你才能成为最强的那把刀」
- 爽点 + 余韵：原来一切都在她的计划中

## 拆文方法总结

从这三个案例中提炼的拆文方法：

1. **标记情绪**：每一段标注虐/爽/蓄力，检查分布是否合理
2. **找钩子**：每篇至少 2 个钩子，至少 1 个信息差钩子
3. **查结构**：起承转合是否完整，转折是否合理
4. **看铺垫**：每个反转前面是否有至少 2 个铺垫线索
5. **评估爽度**：最大爽点是否在全文 70-85% 位置
