# 知乎盐言故事格式规范

## 章节标题

格式：`###第X章`（无标题、无空格、无句号）

```
###第一章
正文内容……

###第二章
正文内容……
```

错误示例：~~### 第一章 破碎~~、~~### 第一章~~、~~第一章 破碎~~

## 段落排版

### 缩进

每段开头两个全角空格：

```
　　她站在门口，看着茶几上那张离婚协议。
　　三年了。
```

### 段间距

- 段落之间：无空行（紧排）
- 章节之间：一个空行

```
　　她站在门口。
　　三年来第一次，她没有哭。
上一章结束

###第二章
　　签字的时候手没抖。
```

## 对话格式

使用中文方括号 `【】`，不用引号：

```
　　【你疯了？】他拍了一下桌子。
　　【我没疯，我只是清醒了。】
　　【清醒的人不会提离婚。】
```

注意：
- 对话独立成段
- 说话人标签（他说/她问）放在对话前面或后面，用句号
- 不用冒号引出对话（~~她说道：「……」~~）

## 场景切换

用数字分隔，独占一行：

```
　　她把钥匙放在鞋柜上。
1
　　法律援助中心的门开着。
```

多场景连续用 1、2、3 递进。

## 省略号

统一用六点：`……`

```
　　【我……】
　　她没说完，因为他已经转身走了……
```

不要用：~~...~~、~~。。。~~、~~………~~

## 视角

仅限第一人称（我）。不要出现第三人称全知视角。

```
✓ 我看着他，忽然觉得陌生。
✗ 她看着他，心里涌起一阵酸涩。
```

## 句式风格

- 短句为主，口语化
- 避免长复句、文言腔、翻译腔
- 一句话一个信息点

```
✓ 他没来。电话也没打。我等了两个小时。
✗ 尽管他并未如约而至，甚至连一通简短的电话也未曾拨来，我仍旧在寒风中固执地等待了漫长的两个小时。
```

## 章节字数

| 类型 | 每章字数 | 说明 |
|------|----------|------|
| 短篇章节 | 400-600 字 | 8000-20000 字总量 |
| 网文章节 | 850-950 字 | 平台连载格式 |

## 章尾收法

以动作或对话结尾，不要总结性叙述：

```
✓ 【你再说一遍。】他笑了，但眼睛没有。
✓ 她拿起杯子，喝了一口已经凉透的咖啡。

✗ 这一天终于结束了，但一切才刚刚开始……
✗ 我心里五味杂陈，不知道明天会怎样。
```

## 完整示例

```
###第一章

　　离婚协议是周一到的。
　　我没签字。不是不想，是手在抖。
　　【你看到桌上那个了吗？】他站在厨房门口。
　　【看到了。】
　　【那你怎么想？】
　　【我想喝杯水。】
　　他没说话。倒了杯水放在我面前。
　　水是凉的。他连烧水都懒得烧了。
1
　　下午我去了一趟银行。
　　柜员问我查哪个账户，我说联名那个。
　　【余额是零。】
　　【不可能。】
　　【确实是零，上月二十三号全部转出。】
　　我看着那张打印单，上面的数字整整齐齐，只有个零。
　　【转到哪里了？】
　　柜员看了我一眼：【对方账户信息需要法院调取。】
　　我点点头，把单子折好放进口袋。
　　走出银行的时候，阳光很好。
　　我掏出手机，拨了一个号码。
　　【喂？律师所吗？我要起诉。】

###第二章

　　律师姓周，四十多岁，说话很慢。
　　【你说全部存款都转走了？】
　　【对。联名账户。】
　　【有其他资产吗？】
　　【一套房子，还在还贷。】
　　他点了点头，在本子上写了几个字。
　　【你这个情况，建议先冻结账户，再申请财产保全。】
　　【他会不会已经把钱藏好了？】
　　周律师摘下眼镜擦了擦：【藏得了钱，藏不了转账记录。】
```
