# 知乎盐言故事格式规范

## 一、YAML Frontmatter

每篇故事开头必须包含 frontmatter：

```yaml
---
title: 故事标题
summary: 一句话概括故事核心（20字以内，用于列表展示）
tags:
  - 标签1
  - 标签2
  - 标签3
publishedAt: 2024-01-01
source: 原创
---
```

**tags 规则**：
- 2-5 个标签
- 优先用平台热门标签：虐恋、重生、复仇、悬疑、反转、治愈、BE、HE、职场、婚姻
- 标签要具体，不要用「小说」「故事」这种泛标签

## 二、章节格式

```
###第1章

正文内容...

###第2章

正文内容...
```

**规则**：
- 用三级标题 `###`
- 格式固定为 `###第X章`，章号用中文数字或阿拉伯数字均可
- 章与章之间空一行
- 每章首句直接进入剧情，不要有章节导语

## 三、段落格式

### 段首缩进

每段开头缩进两个全角空格：

```
　　这是第一段。两个全角空格开头。

　　这是第二段。注意是全角空格，不是半角。
```

**空格规范**：
- 使用全角空格 `　`（U+3000）
- 不是两个半角空格
- 不是 tab 缩进
- 对话段落同样缩进

### 对话格式

使用全角方括号 `【】` 包裹对话：

```
　　【你到底想怎样？】

　　【我不想怎样，我只是想要一个说法。】

　　他沉默了很久，才开口：【那你听好了，我只说一次。】
```

**规则**：
- 对话内容用 `【】` 包裹
- 对话可以独立成段，也可以嵌入叙述中
- 对话动作描写放在括号外
- 不要用 `""` 或 `「」` 或 `「」`

### 对话标点细节

```
　　【我同意。】她点了点头，【但有个条件。】

　　【什么条件？】

　　【让他亲口告诉我。】
```

- 一句对话结束用句号或逗号，跟动作描写连接
- 连续对话省略重复的动作标签
- 避免每句对话都加「他说」「她道」

## 四、分隔线

章节内的场景转换用编号分隔线：

```
————————1————————

　　新场景开始...
```

或：

```
——————1——————

　　新场景开始...
```

**规则**：
- 使用长破折号 + 数字 + 长破折号
- 数字从 1 递增
- 分隔线前后各空一行
- 用于同一章内的场景/时间/视角切换

## 五、叙事视角

**强制第一人称**。

```
✓ 正确：我看到他转过身，眼眶泛红。

✗ 错误：她看到他转过身，眼眶泛红。
```

**例外**：
- 重生文中可以有「上一世」的回忆片段，但叙事仍然是第一人称
- 心理惊悚类可以有叙述者不可靠的情况，但仍是第一人称
- 绝对不用第三人称全知视角

## 六、语言风格

### 短句优先

```
✓ 好：他笑了。我不喜欢那个笑。

✗ 差：他嘴角微微上扬，露出了一个我不太喜欢的笑容。
```

### 口语化

```
✓ 好：我当时脑子嗡的一下。

✗ 差：我的大脑瞬间陷入了短暂的空白状态。
```

**核心原则**：
- 句子不超过 30 字为佳
- 能断句就断句
- 能用口语绝不用书面语
- 形容词越少越好
- 动词 > 形容词 > 副词

### 禁止事项

- 不用「只见」「不禁」「陡然」等古典小说词汇
- 不用大段景物描写
- 不用排比句堆情绪（用细节和动作代替）
- 不用「此时此刻」「就在这时」等过渡词
- 结尾不总结、不升华、不反思

## 七、字数规范

| 类型 | 全文字数 | 每章字数 |
|------|----------|----------|
| 短篇盐言 | 8000-20000 | 400-600 |
| 网文短篇 | 8000-20000 | 850-950 |

**章节字数控制**：
- 短篇盐言：每章 400-600 字，节奏快，一章一个小情绪点
- 网文短篇：每章 850-950 字，每章结尾留钩子

**全文字数分配**（以 12000 字为例）：

| 部分 | 占比 | 字数 |
|------|------|------|
| 开头（钩子） | 5% | 600 |
| 铺垫 | 35% | 4200 |
| 升级 | 25% | 3000 |
| 反转 | 15% | 1800 |
| 结尾 | 20% | 2400 |

## 八、结尾规范

**结尾必须用动作或对话收束，绝对不能用反思或总结。**

```
✓ 正确（动作结尾）：
　　我关上门，把钥匙放在鞋柜上。手机亮了，是他的消息。
　　我没有点开。

✓ 正确（对话结尾）：
　　【那你还爱我吗？】
　　他沉默了很久。
　　【不重要了。】

✗ 错误（反思结尾）：
　　我终于明白，有些错过就是一辈子。人生没有如果...

✗ 错误（总结结尾）：
　　回首这一路，我失去的远比得到的多...

✗ 错误（升华结尾）：
　　窗外的阳光洒进来，照亮了崭新的开始...
```

**结尾原则**：
- 最后一句必须是具体的（动作、对话、画面）
- 留白，让读者自己感受
- 余韵 > 结论
- 好的结尾让读者想翻回开头重看一遍

## 九、格式自检清单

- [ ] frontmatter 完整且格式正确
- [ ] 章节标题格式为 `###第X章`
- [ ] 每段缩进两个全角空格
- [ ] 对话使用 `【】`
- [ ] 场景分隔线使用编号格式
- [ ] 全篇第一人称
- [ ] 无长句（超过 30 字的句子已拆分）
- [ ] 无书面语堆砌
- [ ] 结尾是动作或对话，非反思
- [ ] 字数在目标范围内
