# 网络小说高频概念词典

## 核心术语

| 术语 | 定义 |
|------|------|
| 黄金三章 | 长篇网文前三章（约 1-1.5 万字），决定签约和追读 |
| 爽点 | 让读者产生满足感的情节点，网文的核心商品 |
| 钩子 | 吸引读者继续阅读的悬念或冲突设计 |
| 追读率 | 持续追看更新的读者比例，长篇最核心的指标 |
| 完读率 | 从头读到尾的读者比例，短篇最核心的指标 |
| 铺放比 | 爽点铺垫字数与释放字数的比例，通常 3:1 到 5:1 |
| 日更 | 每天更新，长篇网文的基本要求（4000-6000 字/天） |
| 上架 | 长篇从免费章节转入付费章节（通常 80-120 章） |
| 签约 | 与平台签订合约，通常需要 30-50 章的考核期 |

| 白月光 | 不可替代的初恋/理想对象 |

## 技法术语

| 术语 | 定义 |
|------|------|
| 扮猪吃虎 | 隐藏实力等待时机反击 |
| 装逼打脸 | 展示实力让轻视者震惊 |
| 信息差碾压 | 利用独有信息获得优势 |
| 代入感 | 读者产生「我就是主角」的沉浸感 |
| 情绪拉扯 | 在正负情绪间反复切换吸引读者 |
| 节奏感 | 紧张与舒缓交替的叙事韵律 |
| 知乎体 | 第一人称、短句、口语化的短篇写作风格 |
| 爽文疲劳 | 连续爽点无铺垫导致爽感递减 |
